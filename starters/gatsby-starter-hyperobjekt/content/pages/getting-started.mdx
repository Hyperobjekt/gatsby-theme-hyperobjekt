## Getting Started

You can create a new site from the starter with the following command:

```sh
gatsby new my-new-site https://github.com/Hyperobjekt/gatsby-starter-hyperobjekt
```

### Configuration

Main site configuration is managed through the `/config/metadata.json` file. You can edit it directly, or use the Netlify CMS to edit it.
